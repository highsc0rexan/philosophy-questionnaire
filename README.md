# Philosophy Questionnaire

A short questionnaire that maps your beliefs and decisions to a set of philosophies — Stoicism, Taoism, Existentialism, Optimistic Nihilism, Buddhism, and more.

## Run locally

```bash
pnpm install
pnpm dev
```

Then open the URL it prints (usually http://localhost:5173).

## Build

```bash
pnpm build
```
